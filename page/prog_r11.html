Programação Funcional - Pacote dplyr

Programação funcional com pacote dplyr para organizar, arrumar, corrigir e limpar bases de dados brutas. Além disso, pode ser utilizado para gerar resumos, processar dados e fazer análises. As operações de repetição podem ser feitas de forma fácil, rápida e eficiente utilizando as funções disponibilizadas pelo pacote dplyr. Este pacote tem se tornado cada vez mais popular e facilita muito a vida para quem quer manipular, organizar e analisar dados.
