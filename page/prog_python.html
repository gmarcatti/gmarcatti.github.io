Programação no Python

Olá, seja bem vindo ao curso de Programação no Python! Neste curso você irá aprender a executar operações básicas; trabalhar com objetos; executar e criar funções; usar as estruturas básicas de controle de fluxo: condicional com if-else e repetição com os loops while e for; trabalhar com as principais estruturas de dados básicas disponíveis no Python: list, tuple, dict e set; e estruturas como array e DataFrame disponibilizadas pelos pacotes numpy e pandas, respectivamente; repetição usando a estratégia de split-apply-combine (ou group by) e com os pacotes numpy e pandas; também serão apresentados exemplos de aplicação. Se achar o curso útil inscreva-se no meu canal do YouTube aqui, por favor!
