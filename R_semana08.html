SEMANA 08

Estruturas de dados: Estruturas, Indexação e Loop

< Anterior Próximo >

Você irá aprender sobre as principais estruturas de dados do R: vetor, matriz, lista e data.frame. Você também irá aprender como indexar cada uma dessas estruturas e a utilizá-las em um loop. Apesar de serem específicas do R, podemos encontrar estruturas semelhantes na maioria das linguagens de programação mais modernas.

O vetor já foi amplamente explorado nos tópicos anteriores. Vetor é uma estrutura de dados unidimensional e composto por apenas um tipo de dados. Trata-se de uma estrutura de uso geral, com várias aplicações na implementação de algoritmos.
Matriz é uma estrutura bidimensional, e assim como o vetor suporta apenas um tipo de dado. É amplamente utilizada em operações matemáticas no cálculo.
A data frame permite indexação bidimensional (2d) e também possui o conceito de linhas x colunas, assim como a matriz, porém é mais flexível, no sentido de permitir colunas de tipos de dados diferentes (mas apenas colunas), assim como uma planilha eletrônica, tal como Excel. Essa estrutura é amplamente utilizada atualmente, sobretudo para análises de dados.
No R uma lista é um conjunto de estruturas de dados, sendo o mais comum um conjunto de vetores, isto é, uma lista de vetores. A lista é caracterizada por ser uma estrutura heterogênea, isso quer dizer que além de suportar diferentes tipos de dados, também suporta diferentes tamanhos. Trata-se de uma estrutura de uso geral, é amplamente utilizada para organizar resultados da execução de funções.
